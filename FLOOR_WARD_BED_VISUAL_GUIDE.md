### Occupancy Rate Calculation
```
Formula: (Occupied Beds / Total Beds) Ã— 100

Example:
Total Beds: 110
Occupied: 79
Occupancy: (79 / 110) Ã— 100 = 71.8% â‰ˆ 72%
```

### Critical Occupancy Threshold
```
Warning Level: >85% occupancy
Critical Level: >95% occupancy

System Actions:
- Yellow Alert: 85-94% (Plan discharge/transfers)
- Red Alert: 95-100% (Immediate action required)
```

### Available Beds Calculation
```
Available = Total - (Occupied + Maintenance + Cleaning + Reserved + Blocked)

Example:
Total: 110
Occupied: 79
Maintenance: 3
Cleaning: 5
Reserved: 2
Blocked: 1
Available: 110 - (79+3+5+2+1) = 20
```
